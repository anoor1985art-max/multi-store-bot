# 🏢 المجمع التجاري الذكي متعدد المتاجر (Multi-Store Mall Bot)

بوت تليجرام احترافي ومخصص لإدارة عدة متاجر تجارية مستقلة داخل بوت مركزي واحد على شكل مجمع تجاري (Mall).

## 🛍️ المتاجر المدمجة حالياً في البوت:
1. 📦 **متجر المنتجات الملموسة والتوصيل** (أزياء، عطور، هدايا، مستلزمات منزلية)
2. 💻 **متجر الإلكترونيات والأجهزة الذكية** (هواتف ذكية، لابتوبات، إكسسوارات)
3. 🍔 **متجر الطعام والكافية** (وجبات سريعة، مشرويات، حلويات ومخبوزات مع دعم مشاركة الموقع الجغرافي)

## 🎛️ مميزات النظام:
- **نظام المتاجر المستقلة:** كل متجر يمتلك أقساماً ومنتجات وسلة مشتريات منفصلة بالكامل.
- **إدارة ديناميكية من التليجرام:** يستطيع الإدمن إضافة وحذف وتعديل الأقسام والمنتجات وتغيير أسعارها من داخل البوت مباشرة.
- **إشعارات فورية بالطلب:** يصل إشعار الطلب بالتفاصيل (اسم، هاتف، عنوان أو موقع GPS، المنتجات والمبلغ) فوراً لمدير المتجر المختص.

## 🚀 كيفية التشغيل والربط:
1. افتح ملف `.env` وضع توكن البوت في `TELEGRAM_BOT_TOKEN`.
2. ضع معرفك الشخصي على تليجرام في `SUPER_ADMIN_ID`.
3. لتشغيل البوت على الكمبيوتر، اضغط مزدوجاً على ملف `start_bot.vbs` أو شغل الأمر `python multi_store_bot.py`.
